# @sudocode-ai/better-sqlite3-darwin-arm64

Prebuilt better-sqlite3 native binary for macOS arm64 (Apple Silicon).

This package contains the precompiled native addon for better-sqlite3 v11.10.0, eliminating the need for compilation during installation.

## Installation

This package is installed automatically as an optional dependency when you install `@sudocode-ai/cli` or `@sudocode-ai/local-server` on a compatible platform.

## Platform Support

- **OS**: macOS
- **Architecture**: arm64 (Apple Silicon M1/M2/M3)
- **Node.js**: v18+ (N-API compatible)

## Usage

This package is used internally by sudocode packages. You do not need to import it directly.
